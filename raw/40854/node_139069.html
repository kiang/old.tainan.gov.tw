2014言論自由日系列活動記者會 新聞稿

台南市為紀念鄭南榕先生犧牲生命，爭取百分之百言論自由的事蹟，以4月7日鄭南榕先生殉難的這一天，訂為台南市的「言論自由日」。希望傳承與深化台灣民主自由的價值，讓台灣這片土地，以及人民的心中能開滿民主與自由的花。
今年4月7日言論自由日適逢鄭南榕逝世25週年，台南市政府與鄭南榕基金會合作，以「我主張。NYLON 25」為名舉辦一連串演唱會、藝術展、影展等系列活動，號召民眾對於社會的不公不義勇於說出屬於自己的獨立意志，更重新審視當前台灣社會的民主自由是如何由前人奉獻爭取而來。
活動主軸的「我主張……」，取自於鄭南榕撼動人心的著名號召，這一席話代表著個人對訴求的堅定表態，也象徵嚮往自由民主的具體行動，在當時深深影響了苦無出口的人權鬥士們，更影響後世許多追求民主的人們。轉眼25年過去了，台灣的民主仍在一步步往前突破，當年所爭取的「百分百言論自由」，讓現代的年輕人得以自由的表達自我主張，促進社會在不同議題上的辨證思考。
來到鄭南榕青年時期的台南，這次由藝文界人士運用不同媒介展現南榕的精神和身影，結合音樂、電影、藝術展覽的免費系列活動，包括海安路的搖滾音樂會，遍佈於各人權場所的藝術展覽，深度探討自由人權的電影院等三大活動，使台南市民重溫鄭南榕於台南成長過程中發展出的堅毅民主思想，更帶著各地民眾走進台南，感受南榕精神和言論自由的可貴。

我主張。尋找南榕藝術展
成功大學新廣場的命名爭議，引發討論一個歷史人物所衍生一連串的爭議性事件與言論，使原先建構在歷史性的社會共識再次被審視。藉由藝術的展現，我們將以不同觀點呈現不在同一時空但長存於社會的思想及價值觀，沒有特定場所，更沒有框架，20位藝術家遍佈台南人權象徵場域展現南榕精神，用藝術一同體會南榕、尋找南榕！藉由20位藝術家，以展現言論自由精神創造包括《讚不讚》、《台灣站起來》、《事件‧足跡》、《咱ㄟ心聲》、《狗吠火車》等20件藝術作品在各景點展出。

我主張。獨自行動演唱會
一場充滿音樂性與主張的演唱會！
}   時間：4月12日及13日，下午4點到晚上9點
}   地點：海安路（忠明街至和平街段）
「獨立」、「自由」是Nylon的主張，更是對國家未來的期盼，對於每個人來說，得以自主的追求心中重要價值，更是永不停止的行動。就算在現代來看，鄭南榕追求民主的作為仍然是如此的義無反顧，而此次我主張演唱會策劃統籌，同時也是330震撼凱道之歌《島嶼天光》的創作者楊大正，便是因為欽佩南榕的堅強意志而加入執行團隊，帶著守護台灣的堅定意念，島嶼天光已經成功帶起台灣人捍衛民主的決心，而這樣的力量，也將於此次台南言論自由日展現。
主辦單位特別邀請十組不同樂風的音樂人，橫跨三個世代的演出陣容，包括火燒島、鄭宜農、滅火器、楊烈、大支、打狗亂歌、張睿銓、KoOK、麵包車、閃靈，以演唱的方式大聲喊出民主和言論自由的熱火。

}   地點：中西區全美戲院、鹽水區永成戲院

本影展以「自由時代」為題，重新詮釋媒體在言論自由上的責任，以及激化人民再次爭取個人自由權利的力量，所選擇的影片將以「自由」及「人權」為方向。
